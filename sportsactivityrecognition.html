01
Dataset Preparation
Downloaded UCF-101 from Kaggle Hub. Filtered 10 sports classes (1,456 videos total) from the full 101-class, 13,451-video dataset. Saved filtered train/val/test CSV splits to Google Drive.
02
Spatio-Temporal Feature Extraction
For each video: sample up to 30 frames, compute optical flow between consecutive frames, then extract HOG (shape/appearance), HOF (motion direction), MBH (motion boundary), and trajectory shape features from each frame patch.
03
Temporal Aggregation
Across all sampled frames, compute 4 temporal statistics per descriptor: Mean (μ), Standard Deviation (σ), Max, and Min. Concatenate with histogram vectors → 38,448-dimensional final feature vector.
04
Normalisation & Dimensionality Reduction
StandardScaler applied to zero-mean, unit-variance. PCA reduces 38,448 → 500 orthogonal components capturing maximum variance. This is the same feature vector fed to both ML and NN models.
05
Training & Evaluation
Train 4 traditional models (LR, SVM, RF, XGB) and 4 neural networks (FCNN, RNN, LSTM, GRU) on the same 500D features. Evaluate on held-out test set using Accuracy and F1-score (macro & weighted).

Feature Engineering

What We Extract

APPEARANCE
HOG

Histogram of Oriented Gradients. Captures edge directions and body posture — detects athlete silhouette, limb configuration, equipment shape.

MOTION
HOF

Histogram of Optical Flow. Encodes motion magnitude and direction. Differentiates fast vs. slow throws, swing speed, running vs. standing.

MOTION BOUNDARY
MBH

Motion Boundary Histogram. Gradients of horizontal/vertical flow components. Robust to camera shake. Captures complex multi-joint actions like soccer kicks.

KINEMATICS
Trajectory Shape

Mean + variance of flow vectors across trajectory. Differentiates linear punches vs. curved swings. Encodes the path dynamics of an action.

TEMPORAL STATS
Temporal Features

Min, Max, Mean (μ), Std (σ) computed per descriptor over all frames. Summarises action intensity distribution over time without fixed-length sequences.

# Temporal statistics applied to each descriptor stream def compute_temporal_stats(descriptors): arr = np.array(descriptors) # shape: (T, D) mu = np.mean(arr, axis=0) sd = np.std(arr, axis=0) mx = np.max(arr, axis=0) mn = np.min(arr, axis=0) return np.concatenate([mu, sd, mx, mn]) # 4×D # Final feature vector: HOG + HOF + MBH + Traj + all temporal stats # → 38,448 dims → StandardScaler → PCA(500) → Classifier

All networks use the same 500D PCA features, Adam optimizer, Cross-Entropy loss, batch=32. Shared feature baseline ensures fair comparison.

3 hidden layers: 512→256→128. BatchNorm + Dropout(0.3/0.2). The winner among NNs — sequential structure adds no benefit over flat feature vectors.

2 layers, 512 input → 256 hidden, Dropout=0.5. Best generalisation among RNN variants (overfitting gap only 3.7%). Richer gating helps over GRU here.

GRU

Simplified LSTM with update/reset gates. Fewer params but underperforms LSTM on this small dataset. Update/reset gates not enough to compensate for LSTM's memory cells.

Worst performer. Vanishing gradient problem under small data. Features are aggregated temporal stats, not raw sequential frames — sequential advantage is lost.

Discussion

Key Findings & Analysis

🏆
SVM Wins on Small Data
SVM (70.61%) outperforms the best neural network (FCNN 65.79%) by a 4.82% margin. This challenges the perceived superiority of NNs — with only 1,012 training samples and handcrafted features, SVM's geometric margin-maximisation is more effective.
📐
Linear Separability is Surprisingly Strong
Logistic Regression at 69.74% is virtually tied with SVM, revealing that PCA-reduced handcrafted features are nearly linearly separable. The PCA orthogonal axes directly benefit margin-based classifiers.
🛡
SVM Doesn't Overfit
SVM's overfitting gap is −1.17% (test accuracy actually slightly exceeds best validation). All neural networks overfit more than traditional models. LSTM is the best NN in this regard at only 3.7% gap.
⚠️
Sequential Models Lose Advantage
RNN, LSTM, GRU process data sequentially — but features are already temporally aggregated statistics, not raw sequences. This destroys the temporal ordering advantage and explains their poor performance vs. FCNN.
🎯
LSTM Beats GRU
LSTM (58.8%) outperforms GRU (54.8%). On small datasets, LSTM's richer memory cell architecture with input, forget, and output gates provides more capacity than GRU's simplified update/reset gates.
🏹
Archery/Golf Confusion Problem
Archery remains the most misclassified class, predominantly confused with Golf. Both share standing posture + long object held with one arm + minimal lower body movement. HOG can't distinguish bow from club at these feature resolutions.

Traditional ML · Why It Won

  • PCA features are orthogonal — perfect for SVM margins
  • No gradient descent → no overfitting risk from small batches
  • Logistic regression: high precision + recall balance across all classes
  • Closed-form / convex optimisation avoids local minima
  • Feature normalisation + PCA together benefit ML more than NN

Neural Networks · Why They Struggled

  • Only ~1,000 training samples — NNs need far more
  • Temporal features already aggregated — RNN/LSTM/GRU lose sequential advantage
  • Adam optimizer needs careful LR tuning on small data
  • Dropout helps but can't compensate for insufficient samples
  • Basic architectures — 3D CNN could exploit raw video better
